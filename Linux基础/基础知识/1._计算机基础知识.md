# 进制转换

数字计算机的数制采用二进制，bit 位, byte 字节 1 byte =8 bit ，例如 0101，这里就是4位。只有01010101这样才能组成一个byte；例如：01010101。这个例子中的每一位都是一个比特，而整个序列是一个字节。

-   bc计算器

```bash
[root@litao ~]# bc
bc 1.06.95
Copyright 1991-1994, 1997, 1998, 2000, 2004, 2006 Free Software Foundation, Inc.
This is free software with ABSOLUTELY NO WARRANTY.
For details type `warranty'. 
obase=2
10
1010
#obase=2 以二进制的方式输出，输入10，二进制则是1010

[root@litao ~]# bc
bc 1.06.95
Copyright 1991-1994, 1997, 1998, 2000, 2004, 2006 Free Software Foundation, Inc.
This is free software with ABSOLUTELY NO WARRANTY.
For details type `warranty'. 
ibase=2
10
2
#ibase=2 以二进制的方式输入，十进制输出，输入10，十进制则是2
```
