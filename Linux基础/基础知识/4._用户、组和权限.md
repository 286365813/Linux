# 用户

Linux中每个用户是通过（UID）来唯一标识的

-   管理员：root, 0
-   系统用户：1-499 （CentOS 6以前）, 1-999 （CentOS 7以后） 对守护进程获取资源进行权限分配。
-   登录用户：500+ （CentOS6以前）, 1000+（CentOS7以后） 给用户进行交互式登录使用 。

```shell
[litao@localhost ~]$ id
uid=1000(litao) gid=1000(litao) groups=1000(litao)

[litao@localhost ~]$ getent passwd litao
litao:x:1000:1000::/home/litao:/bin/bash
名称：密码：组ID：UID:：家目录：shell类型

centos默认shell：[litao@localhost ~]$ ll /sbin/nologin 
ubuntu默认shell: litao@ubuntu:~$ ll /bin/false or /usr/bin/nologin
```

# 用户组

Linux中可以将一个或多个用户加入用户组中，用户组是通过Group ID（GID） 来唯一标识的。

-   系统组：1-499（CentOS 6以前）, 1-999（CentOS7以后）, 对守护进程获取资源进行权限分配
-   普通组：500+（CentOS 6以前）, 1000+（CentOS7以后）, 给用户使用

```bash
[root@localhost ~]# getent group | grep litao
litao:x:1000:
```

在使用useradd -g 和 -G时候；-g指的是主要组，-G指的是辅助组。主要组只能是一个；辅助组可有可无，也可以多个。

```bash
[root@localhost ~]# id postfix
uid=89(postfix) gid=89(postfix) groups=89(postfix),12(mail)
```
