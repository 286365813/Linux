# 重定向和管道

`/proc/$BASHID/fd{0、1、2}`

![image.png](assets/重定向和管道-1.png)

输入重定向：描述代码为0；使用 "0<" 或者 "<" 实现标准输入的重定向，其中 0 是可以省略的。

输出重定向：描述代码为；使用 "1 > " 或者 ">" 实现标准输出的重定向，其中1 是可以省略的。

错误重定向： 描述代码为2；使用 "2 >" 或者 "2 >>"实现错误输出重定向，其中2 是可以省略的。

###### 标准输出重定向

通过重定向可以在另外个tty把执行结果显示出来。
